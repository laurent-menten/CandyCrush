# CandyCrush

installer git (https://www.git-scm.com/downloads)

# Pour récupérer le projet:

- ouvrir Git Bash
- aller dans le répertoide ou tu veux mettre le projet (syntax linux)
- git clone https://github.com/laurent-menten/CandyCrush.git

# Pour mettre à jour le projet (si changement sur github)

- ouvrir Git Bash
- aller dans le répertoire du projet
- git pull
